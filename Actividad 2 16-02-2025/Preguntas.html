1. Html

Introduction

  • ¿Qué es HTML?

    HTML es el componente mas básico del internet, es lo que organiza la estructura de una página web y permite acceder a otras páginas web o elementos, el lenguaje utiliza etiquetas para definir lo que se va a mostrar en pantalla, ejemplo: header, imágenes,footer.

  • Explica las etiquetas "head" y "body"

    Head se encarga proveer la información general de los documentos html, provee datos como el título, metadatos, scripts o estilos.

    La etiqueta body representa el contenido del documento, y solo puede haber una etiqueta por documento.

Etiquetas Basicas

  • Explica las etiquetas section,article,aside,header y footer.

    • La etiqueta section representa una sección por separado en un documento.
    • La etiqueta article representa una composición auto-contenida, que se quiere que sea reutilizable de manera independiente.
    • la etiqueta aside es una sección de una pagina donde el contenido esta relacionado contenido principal del documento, se puede utilizar para colocar definiciones.
    • Header representa el contenido introductorio, se utiliza para incluir el título de algo, fecha, etc.
    • La etiqueta footer, representa el pie de página para su ancestro más cercano.

2. CSS

Introducción

  • ¿Qué es CSS?

    CSS o Cascading Style Sheets, es el lenguaje de estilos para representar documentos html o XML, es lo que le agrega el diseño a los elementos, como color, tamaño, fuente, posición, etc.

  • ¿Cuál es el proposito?

    Añadir diseño a una página.

  • ¿Como aplicas CSS a un documento html

    • Utilizando style al principio del documento.
    • Añadiendo una referencia a un documento .css.
    • Utilizando style directamente en un elemento.

  • ¿Que es un selector? que es una regla en CSS?

    Un selector indica que reglas, es decir, propiedades, se le van a aplicar a uno o más elementos en específico.

Basic CSS

  • Tag Selector

    Selecciona todos los elementos que compartan etiqueta.

  • Class Selector

    Selecciona todos los elementos de una clase. ejemplo: .nombredeclase

  • Tag and Class Selector

    Selecciona los elementos que compartan la misma tag y clase. ejemplo: tag.nombredeclase

  • Id Selector

    Selecciona el elemento con el id especificado. ejemplo: #id

  • ¿Qué es una clase? ¿Qué es un id? ¿Cómo se relacionan con CSS?

    Una clase es un grupo de elementos, y un id es un indicador único para cada elemento. Estos estan relacionados con css ya que nos ayudan a definir reglas especificas para este grupo de elementos
